## Deskripsi
Pojok Kafe adalah aplikasi kasir berbasis web yang dibangun menggunakan PHP dan MySQL. Sistem ini digunakan untuk mengelola transaksi, produk, kategori, karyawan, serta laporan penjualan.

## Teknologi
- PHP Native
- MySQL
- HTML
- CSS (Tailwind CSS)
- JavaScript

## Fitur
- Login multi role (Owner dan Karyawan)
- Manajemen produk
- Manajemen kategori
- Manajemen karyawan
- Transaksi kasir
- Laporan penjualan
- Profil pengguna
